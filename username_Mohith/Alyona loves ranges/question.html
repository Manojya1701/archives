Problem: Alyona Loves Ranges!

Alyona always plays with the numbers and the numbers that are between two numbers. Alyona's teacher notices this thing and gives her a mini assesment in that, The teacher will give her a number N and two more number L and R. Alyona has to find the number of numbers between L and R. And asks her to find the least possible number which is the range L & R, let's call that number as 'x' and the find gcd(n,x) such that that number is also in the range [L,R] As alyona is little busy, she needs your help to solve this problem!

Input Specification:

  • input1: The number of test cases t (1 <= t <= 105)
  • input2: The number that is given by teacher N (1 <= N <= 109).
  • input3: The start and end numbers in the range L,R (1 <= L,R <= 109).

Output Specification:

Print the number which satisifies the conditions mentioned, If there is not valid number, print -1

Example

Input

        4
        6 5 7
        16 1 6
        10 1 10
        2 3 4

Output

        6
        1
        1
        -1

Explanation

  • In test case 1, The number 6 is the least number in the range [5,7] and gcd(6,6) = 6 which is also in the range [5,7]
  • In test case 2, The number 1 is the least number in the range [1,6] and gcd(16,1) = 1 which is also in the range [1,6]
  • In test case 4, The number 2 doesn't satisfy the condition because gcd(2,3) = 1 and gcd(2,4) = 2, but 2 is not in the range [3,4]. Hence, the output is -1.
